## Overview

VaultWares releases software and firmware updates regularly. Updates may include security patches, bug fixes, new features, and performance improvements. Staying current with updates is an important part of your security posture.

## Update channels

| Channel | Description | Recommended for |
|---------|-------------|-----------------|
| **Stable** | Thoroughly tested releases, released approximately quarterly | All production environments |
| **LTS** | Long-term support with 5-year security patch commitment | Regulated environments with infrequent change windows |
| **Security** | Out-of-band patches for critical security vulnerabilities | All environments — prioritize immediately |
| **Beta** | Pre-release testing | Lab and non-production environments only |

## How to check for updates

```bash
# Check all managed products for available updates
vw-admin updates check --all --format table

# Output:
# Product          Current     Available   Channel  Severity   Release date
# VaultCrypt       3.1.2       3.2.0       Stable   Security   2024-11-01
# VaultAccess      3.1.5       3.2.0       Stable   Feature    2024-11-01
# VaultBackup      3.0.8       3.2.0       Stable   Feature    2024-11-01
# VaultMonitor     3.1.1       3.2.0       Stable   Feature    2024-11-01
# VaultDrive FW    3.1.2       3.2.0       Stable   Security   2024-10-15
# VaultHSM FW      5.4.1       5.5.0       Stable   Feature    2024-10-20
# VaultGate FW     4.2.3       4.3.0       Stable   Security   2024-11-05

# Subscribe to security advisories via email
vw-admin updates subscribe \
  --email security@example.com \
  --channels security,stable \
  --products all
```

## Software update procedures

### Automatic update notifications

```bash
# Configure automatic update checks and email notifications
vw-admin updates configure \
  --check-frequency daily \
  --notify-email security@example.com \
  --auto-download true \
  --auto-install false \  # Review before installing in production
  --channel stable
```

### Manual update (Linux)

```bash
# Update VaultCrypt
sudo apt-get update && sudo apt-get install --only-upgrade vaultcrypt-enterprise

# Update VaultAccess
sudo apt-get update && sudo apt-get install --only-upgrade vaultaccess

# Verify installed versions
vaultcrypt --version
vw-admin --version
vw-monitor --version
vw-backup --version
```

### Kubernetes rolling update

```bash
# Update all VaultWares deployments to latest stable
for deployment in vaultaccess vaultbackup vaultmonitor; do
  kubectl set image deployment/$deployment \
    $deployment=registry.vaultwares.com/$deployment:3.2.0 \
    -n vaultwares
  kubectl rollout status deployment/$deployment -n vaultwares
done
```

## Security advisories

Subscribe to receive security advisories by email:  
[portal.vaultwares.com/security-advisories](https://portal.vaultwares.com/security-advisories)

Critical security updates (CVSS 7.0+) are announced via:
- Email to all registered product owners
- Portal banner notification
- RSS feed: `https://security.vaultwares.com/advisories.rss`

VaultWares follows [responsible disclosure](https://security.vaultwares.com/disclosure) and coordinates with researchers through [security@vaultwares.com](mailto:security@vaultwares.com).

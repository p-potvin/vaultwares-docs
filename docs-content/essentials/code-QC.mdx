## Code en ligne

Pour désigner un `mot` ou une `phrase` comme du code, encadrez-le avec des accents graves (`` ` ``).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

## Blocs de code

Utilisez les [blocs de code délimités](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en encadrant le code avec trois accents graves et suivez les accents graves de tête avec le langage de programmation de votre extrait pour obtenir une coloration syntaxique. Optionnellement, vous pouvez également écrire le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

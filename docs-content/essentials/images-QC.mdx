## Image

### Avec Markdown

La [syntaxe Markdown](https://www.markdownguide.org/basic-syntax/#images) vous permet d'ajouter des images à l'aide du code suivant

```md
![title](/path/image.jpg)
```

Notez que la taille du fichier image doit être inférieure à 5 Mo. Sinon, nous vous recommandons de l'héberger sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Vous pourrez ensuite utiliser cette URL pour l'intégrer.

### Avec des intégrations

Pour obtenir plus de personnalisation avec les images, vous pouvez également utiliser [intégrations](/writing-content/embed) pour ajouter des images

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Charge une autre page HTML à l'intérieur du document. Le plus souvent utilisé pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```

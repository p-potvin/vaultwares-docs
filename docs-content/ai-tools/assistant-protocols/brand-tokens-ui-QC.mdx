# Interface utilisateur des jetons de marque

- ID de catégorie : `BRAND_TOKENS_UI`
- Source du résumé : `instructions/summaries/BRAND_TOKENS_UI.md`
- Source des notes : `instructions/notes/BRAND_TOKENS_UI.md`

S'applique lorsque : UI/UX, style, mise en page, image de marque, thèmes, jetons, icônes, règles visuelles.
Faire :
- Utiliser le système de conception VaultWares (mis à jour le 17 mai 2026) comme SoT.
- Lisez et suivez :
  - thèmes-vaultwares/AGENTS.md
  - vaultwares-themes/brand/ui-kit.md
  - vaultwares-themes/brand/tokens/tokens.ts
  - vaultwares-themes/brand/tokens/tailwind.config.ts
- Ne jamais coder en dur les couleurs, polices, espacements, rayons, mouvements réutilisables ; utilisez des jetons nommés.
Pas :
- Ajouter une nouvelle interface utilisateur sans utiliser de jeton.
Fais quand :
- Les changements visuels s'expriment à l'aide de jetons et suivent les règles de la marque.

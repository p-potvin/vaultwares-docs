# Interface utilisateur des jetons de marque

- ID de catégorie : `BRAND_TOKENS_UI`
- Source du résumé : `instructions/summaries/BRAND_TOKENS_UI.md`
- Source des notes : `instructions/notes/BRAND_TOKENS_UI.md`

S'applique lorsque : UI/UX, style, mise en page, marque, thèmes, jetons, icônes, règles visuelles.
À faire :
- Utiliser le système de conception VaultWares (mis à jour le 17 mai 2026) comme source de vérité (SoT).
- Lire et suivre :
  - vaultwares-themes/AGENTS.md
  - vaultwares-themes/brand/ui-kit.md
  - vaultwares-themes/brand/tokens/tokens.ts
  - vaultwares-themes/brand/tokens/tailwind.config.ts
- Ne jamais coder en dur les couleurs, polices, espacements, rayons ou animations réutilisables ; utiliser des jetons nommés.
Ne pas :
- Ajouter de nouvelle interface utilisateur sans utiliser de jetons.
Terminé lorsque :
- Les changements visuels sont exprimés via des jetons et suivent les règles de la marque.

## Notes
Les assistants lisent les notes complètes uniquement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/BRAND_TOKENS_UI.md`

# Politique d'exportation de sauvegarde

- ID de catégorie : `BACKUP_EXPORT_POLICY`
- Source du résumé : `instructions/summaries/BACKUP_EXPORT_POLICY.md`
- Source des notes : `instructions/notes/BACKUP_EXPORT_POLICY.md`

S'applique lorsque : sauvegardes, exportations, migrations, création d'archives, planification de restauration.
Faire :
- Définir ce qui est inclus (données + métadonnées + blocage) et comment restaurer.
- Préférez un test de restauration aux affirmations « sauvegarde réussie ».
Pas :
- Produisez des sauvegardes partielles sans indiquer d'exclusions.
Fais quand :
- La sauvegarde est restaurable et le chemin de restauration est documenté.

## Remarques
Les assistants ne lisent les notes complètes que lorsqu'ils y sont explicitement invités : « lire les notes complètes ».
Chemin des notes : `instructions/notes/BACKUP_EXPORT_POLICY.md`

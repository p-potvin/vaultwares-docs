# Synchronisation globale des instructions

- ID de catégorie : `GLOBAL_INSTRUCTION_SYNC`
- Source de résumé : `instructions/summaries/GLOBAL_INSTRUCTION_SYNC.md`
- Source de notes : `instructions/notes/GLOBAL_INSTRUCTION_SYNC.md`

S'applique lorsque : on modifie `vaultwares-docs/AGENTS.md` (instructions globales de niveau 1).
À faire :
- Après avoir modifié `vaultwares-docs/AGENTS.md`, exécuter : `vaultwares-docs/scripts/sync-global-instructions.ps1`
- Synchroniser uniquement les pré-instructions/pointeurs ; ne pas dupliquer la bibliothèque de protocole complète dans les cibles hôtes.
À ne pas faire :
- Laisser les cibles hôtes obsolètes après les changements dans `AGENTS.md`.
Terminé lorsque :
- La synchronisation a réussi (ou qu'un échec et la raison sont enregistrés).

## Notes
Les assistants lisent les notes complètes uniquement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/GLOBAL_INSTRUCTION_SYNC.md`

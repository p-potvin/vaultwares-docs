# Modifications de fichiers

- ID de catégorie : `FILE_CHANGES`
- Source du résumé : `instructions/summaries/FILE_CHANGES.md`
- Source des notes : `instructions/notes/FILE_CHANGES.md`

S'applique quand : création/déplacement/suppression/renommage de fichiers ou de dossiers.
Faire :
- Garder les changements minimaux et intentionnels; éviter les changements massifs.
- Lors de la suppression, confirmer que les références sont retirées et que les constructions/tests passent toujours.
- Utiliser des chemins sûrs et explicites; éviter les suppressions par joker.
Ne pas :
- Supprimer des chemins ambigus sans confirmer la portée.
Terminé quand :
- Les constructions/tests/vérifications du dépôt réussissent et qu'aucun lien mort ne subsiste.

## Notes
Les assistants lisent les notes complètes seulement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/FILE_CHANGES.md`

# Assistant protocols

This section mirrors `vaultwares-docs/instructions/summaries/*` for browsing.
Notes are human reference and are read by assistants only when explicitly prompted: `read full notes`.

## Routines (always)
1) Safety+scope gate: ask clarifying questions when unsure or risky.
2) Routing: scan all protocol categories; select relevant; open summaries in category order.
3) Tools/routines: run MCP routines only if useful.
4) Ledger: record work as the last step before replying.

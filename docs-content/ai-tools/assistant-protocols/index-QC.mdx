# Protocoles des assistants

Cette section est un miroir de `vaultwares-docs/instructions/summaries/*` pour la navigation.
Les notes sont une référence humaine et ne sont lues par les assistants que lorsqu'on leur demande explicitement : `read full notes`.

## Routines (toujours)
1) Porte de sécurité et de portée : poser des questions de clarification quand on n'est pas sûr ou que c'est risqué.
2) Routage : analyser toutes les catégories de protocoles; sélectionner les pertinentes; ouvrir les résumés par ordre de catégorie.
3) Outils/routines : exécuter les routines MCP seulement si c'est utile.
4) Grand livre (Ledger) : enregistrer le travail comme dernière étape avant de répondre.

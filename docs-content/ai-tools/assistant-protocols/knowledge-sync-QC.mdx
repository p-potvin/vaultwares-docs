# Synchronisation des connaissances

- ID de catégorie : `KNOWLEDGE_SYNC`
- Source de résumé : `instructions/summaries/KNOWLEDGE_SYNC.md`
- Source de notes : `instructions/notes/KNOWLEDGE_SYNC.md`

S'applique quand : les règles dérivent, des conseils récurrents sont répétés, ou qu'un guide de référence durable est nécessaire.
À faire :
- Préférer mettre à jour le résumé de protocole canonique (instructions/summaries) plutôt que de répéter des conseils ad hoc.
- Pour les découvertes « coûteuses », mettre à jour KNOWLEDGE_SCOUT.md.
À ne pas faire :
- Disperser les « connaissances tribales » dans des commentaires de PR aléatoires seulement.
Terminé quand :
- La source durable est mise à jour et liée.

## Notes
Les assistants lisent les notes complètes seulement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/KNOWLEDGE_SYNC.md`

# Normes de documentation

- ID de catégorie : `DOCS_STANDARDS`
- Source du résumé : `instructions/summaries/DOCS_STANDARDS.md`
- Source des notes : `instructions/notes/DOCS_STANDARDS.md`

S'applique lorsque : l'ajout/la modification de pages de contenu de documents ou la navigation dans des documents.
Faire :
- Chaque page doit avoir une couverture YAML avec un titre et une description.
- Utilisez la voix à la deuxième personne ; liens internes relatifs ; balises de langue sur les blocs de code.
- Ajouter de nouvelles pages à la navigation docs.json.
Pas :
- Créer des pages orphelines hors navigation.
Fais quand :
- La page s'affiche, l'entrée de navigation existe et la vérification est enregistrée.

## Remarques
Les assistants ne lisent les notes complètes que lorsqu'ils y sont explicitement invités : « lire les notes complètes ».
Chemin des notes : `instructions/notes/DOCS_STANDARDS.md`

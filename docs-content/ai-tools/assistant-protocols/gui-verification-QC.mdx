# Vérification de l'interface graphique

- ID de catégorie : `GUI_VERIFICATION`
- Source du résumé : `instructions/summaries/GUI_VERIFICATION.md`
- Source des notes : `instructions/notes/GUI_VERIFICATION.md`

S'applique quand : interface web, site de documentation, tableaux de bord, panneaux d'administration, SPA.
À faire :
- Naviguer sur au moins 2 liens de profondeur et revenir en arrière.
- Vérifier la console et le réseau des outils de développement pour les erreurs bloquantes.
- Déclencher les états de survol/focus et un état vide/d'erreur pertinent au changement.
- Vérifier avec une deuxième modalité si possible (curl/Invoke-WebRequest ou Playwright).
À ne pas faire :
- Marquer comme « fonctionnel » à partir d'une seule capture d'écran ou d'un 200 OK.
Terminé quand :
- Vous pouvez lister les pages visitées et ce que vous avez inspecté.

## Notes
Les assistants lisent les notes complètes seulement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/GUI_VERIFICATION.md`

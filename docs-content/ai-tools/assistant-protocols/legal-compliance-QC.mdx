# Conformité légale

- ID de catégorie : `LEGAL_COMPLIANCE`
- Source du résumé : `instructions/summaries/LEGAL_COMPLIANCE.md`
- Source des notes : `instructions/notes/LEGAL_COMPLIANCE.md`

S'applique quand : exigences de conformité, contraintes légales, application des politiques.
À faire :
- Quand on n'est pas sûr, on demande et on ne devine pas.
- Préférer des affirmations minimales; éviter de promettre des certifications/garanties sans preuve.
À ne pas faire :
- Donner des conseils juridiques comme une certitude.
Terminé quand :
- Les contraintes sont explicites et reflétées dans le plan de mise en œuvre.

## Notes
Les assistants lisent les notes complètes seulement quand on leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/LEGAL_COMPLIANCE.md`

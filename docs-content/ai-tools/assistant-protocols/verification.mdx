# Verification

- Category ID: `VERIFICATION`
- Summary source: `instructions/summaries/VERIFICATION.md`
- Notes source: `instructions/notes/VERIFICATION.md`

Applies when: claiming anything works or is complete.
Do:
- Prefer the smallest verification that proves the change.
- Record what was actually run/checked (commands, pages visited, outputs).
Do not:
- Claim “working” from a single 200 OK.
Done when:
- Verification evidence is recorded and repeatable.

## Notes
Assistants read full notes only when explicitly prompted: `read full notes`.

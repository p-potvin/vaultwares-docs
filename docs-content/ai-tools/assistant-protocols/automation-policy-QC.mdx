# Politique d'automatisation

- ID de catégorie : `AUTOMATION_POLICY`
- Source du résumé : `instructions/summaries/AUTOMATION_POLICY.md`
- Source des notes : `instructions/notes/AUTOMATION_POLICY.md`

S'applique lorsque : cron, moniteurs, automatisations récurrentes, tâches en arrière-plan, planificateurs.
Faire :
- Préférez les configurations explicites et inspectables ; éviter l'état caché.
- Sauvegardez ce qui s'exécute où et comment arrêter/annuler.
Pas :
- Créez des automatisations qui s'exécutent en dehors de la politique tailnet.
Fais quand :
- L'automatisation est documentée et vérifiable.

## Remarques
Les assistants ne lisent les notes complètes que lorsqu'ils y sont explicitement invités : « lire les notes complètes ».
Chemin des notes : `instructions/notes/AUTOMATION_POLICY.md`

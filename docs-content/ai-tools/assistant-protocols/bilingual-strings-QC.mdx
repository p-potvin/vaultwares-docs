# Cordes bilingues

- ID de catégorie : `BILINGUAL_STRINGS`
- Source du résumé : `instructions/summaries/BILINGUAL_STRINGS.md`
- Source des notes : `instructions/notes/BILINGUAL_STRINGS.md`

S'applique lorsque : toute chaîne destinée à l'utilisateur, étiquette d'interface utilisateur, navigation, copie marketing, messages d'erreur.
Faire :
- Fournir les versions anglaise (EN) et française québécoise (QC).
- Conception pour les chaînes QC 15 à 20 % plus longues que EN.
Pas :
- Expédiez les nouvelles chaînes destinées aux utilisateurs en EN uniquement.
Fais quand :
- EN et QC sont présents et la mise en page est tolérante.

## Remarques
Les assistants ne lisent les notes complètes que lorsqu'ils y sont explicitement invités : « lire les notes complètes ».
Chemin des notes : `instructions/notes/BILINGUAL_STRINGS.md`

# Chaînes bilingues

- ID de catégorie : `BILINGUAL_STRINGS`
- Source de résumé : `instructions/summaries/BILINGUAL_STRINGS.md`
- Source de notes : `instructions/notes/BILINGUAL_STRINGS.md`

S'applique quand : toute chaîne visible par l'utilisateur, étiquette d'interface utilisateur, navigation, contenu marketing, messages d'erreur.
Faire :
- Fournir les versions anglais (EN) et français québécois (QC).
- Concevoir en tenant compte du fait que les chaînes QC sont 15 à 20 % plus longues que les EN.
Ne pas faire :
- Livrer de nouvelles chaînes visibles par l'utilisateur en EN seulement.
Complété quand :
- Les versions EN et QC sont présentes et la mise en page est tolérante.

## Notes
Les assistants lisent les notes complètes seulement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/BILINGUAL_STRINGS.md`

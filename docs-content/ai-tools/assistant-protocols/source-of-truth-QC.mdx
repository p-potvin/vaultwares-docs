# Source de Vérité

- ID de catégorie : `SOURCE_OF_TRUTH`
- Source de résumé : `instructions/summaries/SOURCE_OF_TRUTH.md`
- Source de notes : `instructions/notes/SOURCE_OF_TRUTH.md`

S'applique quand : toute tâche VaultWares; n'importe quand vous n'êtes pas sûr de savoir où chercher.
À faire :
- Identifier le niveau de gouvernance pour le changement (Tier1 vaultwares-docs; Tier2 vaultwares-themes, vaultwares-adk, agent-ledger; Tier3 par dépôt).
- Préférer le fichier SoT le plus précis (dans le périmètre) plutôt que de chercher dans tout le dépôt.
- Si un fichier SoT référencé est manquant, le signaler et proposer l'emplacement canonique le plus approprié; ne pas inventer de nouveaux chemins SoT en silence.
À ne pas faire :
- Dupliquer les protocoles globaux dans des dépôts aléatoires.
Terminé quand :
- Vous pouvez nommer les fichiers SoT exacts que vous avez suivis et pourquoi.

## Notes
Les assistants lisent les notes complètes seulement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/SOURCE_OF_TRUTH.md`

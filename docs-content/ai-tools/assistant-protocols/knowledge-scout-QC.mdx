# Knowledge Scout

- ID de catégorie : `KNOWLEDGE_SCOUT`
- Source de résumé : `instructions/summaries/KNOWLEDGE_SCOUT.md`
- Source de notes : `instructions/notes/KNOWLEDGE_SCOUT.md`

S'applique lorsque : le débogage coûteux, les particularités difficiles à trouver, ou la découverte répétée.
Faire :
- Consulter vaultwares-docs/KNOWLEDGE_SCOUT.md lorsque l'initialisation ou la recherche SoT échoue, ou pour des tâches d'opérations coûteuses.
- Ajouter une ligne lorsque vous enregistrez plus de 3 appels d'outils ou plus de 5 minutes de recherche.
Ne pas faire :
- Mettre des secrets dans le rapport de reconnaissance.
Terminé lorsque :
- L'entrée de reconnaissance est concise et exploitable.

## Notes
Les assistants lisent les notes complètes uniquement lorsqu'on le leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/KNOWLEDGE_SCOUT.md`

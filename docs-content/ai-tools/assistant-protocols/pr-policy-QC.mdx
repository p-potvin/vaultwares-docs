# Politique de PR

- ID de catégorie : `PR_POLICY`
- Source de résumé : `instructions/summaries/PR_POLICY.md`
- Source de notes : `instructions/notes/PR_POLICY.md`

S'applique quand : création ou mise à jour des PR, demande de révision, application par CI.
À faire :
- Maintenir le périmètre de la PR restreint ; inclure les étapes de vérification et les preuves.
- Ajouter les réviseurs requis spécifiés par la politique du dépôt.
À ne pas faire :
- Fusionner directement dans `main` sauf instruction explicite.
Terminé quand :
- La PR est publiée et inclut ce qui a changé, comment c'est vérifié, et les risques restants.

## Notes
Les assistants lisent les notes complètes seulement quand on les leur demande explicitement : `read full notes`.
Chemin des notes : `instructions/notes/PR_POLICY.md`

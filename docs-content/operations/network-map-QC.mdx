## Modèle d’accès

- **Chiffrement réseau:** Tailscale (WireGuard).
- **TLS public:** nginx + Let’s Encrypt sur les VPS.
- **SSH:** **Seulement sur le Tailnet** (port `22` autorisé uniquement sur `tailscale0`).
- **RDP:** Seulement sur le Tailnet vers `Clopeux-Desktop:3389` (contrôlé par la politique).

## Liste d’intégration

1. Rejoindre le tailnet (client Tailscale).
2. Confirmer que votre appareil apparaît dans la console d'administration et est autorisé.
3. Lire cette page et le document de politique Tailscale.
4. Obtenir l'accès à VaultWarden (`https://warden.vaultwares.ca`) (liste blanche tailnet + autorisation de service).
5. Vérifier :
   - Vous pouvez atteindre `docs.vaultwares.ca` depuis le tailnet.
   - Vous pouvez atteindre `vaultwares.ca` publiquement.
   - Vous pouvez atteindre le point d'accès API PC nécessaire pour votre travail (si la politique le permet).

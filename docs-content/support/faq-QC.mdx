## Général
<AccordionGroup>
  <Accordion title="Qu'est-ce que VaultWares?">
    VaultWares fournit des solutions de protection et de sécurité des données de niveau entreprise, incluant des dispositifs de stockage chiffrés, des modules de sécurité matériels (HSM), des dispositifs d'authentification biométrique, des appareils de sécurité réseau et des logiciels de chiffrement.
  </Accordion>

  <Accordion title="Quel produit est fait pour moi?">
    - **Petite entreprise**: VaultDrive Pro + VaultCrypt Professional
    - **Entreprise**: VaultHSM + VaultGate + VaultCrypt Enterprise
    - **Utilisateurs individuels**: VaultDrive Pro + VaultCrypt Standard
    - **Développeurs**: VaultHSM + accès API

    Contactez notre [équipe de vente](https://vaultwares.com/contact) pour des recommandations personnalisées.
  </Accordion>

  <Accordion title="Offrez-vous des essais gratuits?">
    Oui, nous offrons des essais gratuits de 30 jours pour tous les produits logiciels. Les produits matériels peuvent être évalués par notre programme de démonstration. Contactez sales@vaultwares.com pour plus de détails.
  </Accordion>

  <Accordion title="Quelles certifications ont vos produits?">
    Nos produits sont certifiés pour :
    - FIPS 140-2 Niveau 3 et FIPS 140-3 Niveau 4
    - Common Criteria EAL4+ et EAL5+
    - ISO 27001
    - SOC 2 Type II
    - FBI PIV-071006 (dispositifs biométriques)
  </Accordion>
</AccordionGroup>

## Questions techniques
<AccordionGroup>
  <Accordion title="Quels algorithmes de chiffrement utilisez-vous?">
    - **Symétriques**: AES-256-XTS, AES-256-GCM
    - **Asymétriques**: RSA-2048/4096, ECC (P-256, P-384, P-521)
    - **Hachage**: SHA-256, SHA-384, SHA-512
    - **Dérivation de clés**: PBKDF2, HKDF

    Tous les algorithmes sont validés FIPS 140-2.
  </Accordion>

  <Accordion title="Puis-je utiliser les produits VaultWares dans des environnements virtuels?">
    Oui, nos produits logiciels prennent entièrement en charge :
    - VMware vSphere 7.0+
    - Microsoft Hyper-V
    - KVM
    - Conteneurs Docker

    Note : Les HSM nécessitent un passage PCIe ou des modèles attachés au réseau pour les environnements virtualisés.
  </Accordion>

  <Accordion title="Supportez-vous les déploiements dans le cloud?">
    Oui, les produits VaultWares fonctionnent sur :
    - Amazon Web Services (AWS)
    - Microsoft Azure
    - Google Cloud Platform (GCP)
    - Environnements cloud privés

    VaultHSM Cloud est spécifiquement conçu pour les déploiements dans le cloud.
  </Accordion>

  <Accordion title="Quels systèmes d'exploitation sont pris en charge?">
    - **Windows**: 10, 11, Server 2016+
    - **macOS**: 11 (Big Sur) et ultérieur
    - **Linux**: Ubuntu 20.04+, RHEL 8+, CentOS 8+, Debian 11+

    Consultez [les exigences système](/getting-started/system-requirements) pour plus de détails.
  </Accordion>

  <Accordion title="Puis-je intégrer VaultWares avec mon infrastructure existante?">
    Oui, nous prenons en charge l'intégration avec :
    - Active Directory / LDAP
    - SAML / OAuth / OpenID Connect
    - Systèmes SIEM (Splunk, QRadar, ArcSight)
    - Infrastructure PKI
    - Systèmes de contrôle d'accès physique

    Consultez nos [guides d'intégration](/installation/integration-guides) pour plus de détails.
  </Accordion>
</AccordionGroup>

## Sécurité et conformité
<AccordionGroup>
  <Accordion title="Mes données sont-elles sécurisées?">
    - Toutes les données sont chiffrées avec le chiffrement AES-256
    - Les clés sont stockées dans un matériel résistant aux altérations
    - Authentification multi-facteurs disponible
    - Audits de sécurité et tests d'intrusion réguliers
    - Architecture de connaissance zéro (nous ne pouvons pas accéder à vos données)
  </Accordion>

  <Accordion title="Êtes-vous conforme au RGPD?">
    Oui, les produits VaultWares sont conçus pour vous aider à respecter les exigences du RGPD :
    - Chiffrement des données (Article 32)
    - Droit à l'effacement (Article 17)
    - Portabilité des données (Article 20)
    - Confidentialité dès la conception (Article 25)

    Consultez notre [guide de conformité RGPD](/compliance/gdpr) pour plus de détails.
  </Accordion>

  <Accordion title="Êtes-vous conforme à la HIPAA?">
    Oui, les produits VaultWares répondent aux exigences de la règle de sécurité HIPAA pour :
    - Chiffrement des ePHI au repos et en transit
    - Contrôles d'accès et journalisation des audits
    - Contrôles d'intégrité
    - Sécurité de transmission

    Consultez notre [guide de conformité HIPAA](/compliance/hipaa) pour plus de détails.
  </Accordion>

  <Accordion title="Que se passe-t-il si j'oublie mon mot de passe?">
    - **Dispositifs matériels**: Contactez le support avec une preuve de propriété. Les données ne peuvent pas être récupérées sans le mot de passe en raison du chiffrement matériel.
    - **Logiciel**: Utilisez la clé de récupération générée lors de la configuration
    - **Entreprise**: Les administrateurs peuvent réinitialiser les mots de passe via la console de gestion
    - **HSM**: L'Agent de sécurité peut réinitialiser les PIN des utilisateurs
  </Accordion>

  <Accordion title="VaultWares peut-il accéder à mes données chiffrées?">
    Non. VaultWares utilise le chiffrement de connaissance zéro. Nous n'avons pas accès à vos clés de chiffrement ni à vos données. Seul vous pouvez déchiffrer vos données.
  </Accordion>
</AccordionGroup>

## Questions spécifiques aux produits
### Dispositifs de stockage chiffrés
<AccordionGroup>
  <Accordion title="Puis-je utiliser VaultDrive sur plusieurs ordinateurs?">
    Oui, les dispositifs VaultDrive fonctionnent sur n'importe quel ordinateur avec un port USB. Aucune installation logicielle n'est requise.
  </Accordion>

  <Accordion title="Que se passe-t-il si j'entre le mauvais mot de passe trop de fois?">
    Après 10 tentatives infructueuses, le dispositif se verrouille pendant 30 minutes. Après 20 tentatives infructueuses, le dispositif effectue une effacement cryptographique de clé (toutes les données sont perdues de façon permanente).
  </Accordion>

  <Accordion title="Puis-je récupérer les données si le dispositif tombe en panne?">
    Si le dispositif tombe en panne mécaniquement, les données peuvent être récupérées par notre service de récupération de données. Si la puce de chiffrement tombe en panne, les données ne peuvent pas être récupérées. Conservez toujours des sauvegardes des données critiques.
  </Accordion>
</AccordionGroup>

### Modules de sécurité matériels
<AccordionGroup>
  <Accordion title="Quelle est la différence entre les HSM PCIe et les HSM réseau?">
    - **HSM PCIe**: Installé à l'intérieur d'un serveur, latence plus faible, accès à un seul serveur
    - **HSM Réseau**: Appareil autonome, accessible par plusieurs serveurs, plus facile à regrouper (clusteriser)
  </Accordion>

  <Accordion title="Puis-je regrouper (clusteriser) des HSM pour une haute disponibilité?">
    Oui, VaultHSM prend en charge le regroupement actif-actif et actif-passif avec basculement automatique.
  </Accordion>

  <Accordion title="Comment sauvegarder les clés des HSM?">
    Utilisez le fractionnement de clés M-sur-N pour créer des sauvegardes sécurisées réparties entre plusieurs dépositaires. Consultez [la documentation HSM](/hardware/hsm) pour plus de détails.
  </Accordion>
</AccordionGroup>

### Dispositifs biométriques
<AccordionGroup>
  <Accordion title="Quelle est la précision de vos dispositifs biométriques?">
    - **Taux de fausse acceptation (TFA)**: < 0,001 % (empreinte digitale), < 0,0001 % (visage)
    - **Taux de faux rejet (TFR)**: < 0,1 % (empreinte digitale), < 0,1 % (visage)
  </Accordion>

  <Accordion title="Les données biométriques peuvent-elles être volées?">
    Non. Nous stockons uniquement des modèles mathématiques, pas les empreintes digitales ou les images faciales réelles. Les modèles sont chiffrés et ne peuvent pas être rétro-conçus.
  </Accordion>

  <Accordion title="Que se passe-t-il si mon empreinte digitale change (blessure, vieillissement)?">
    Enregistrez plusieurs doigts lors de la configuration. Vous pouvez également vous ré-enregistrer à tout moment pour mettre à jour votre modèle biométrique.
  </Accordion>
</AccordionGroup>

### Appareils réseau
<AccordionGroup>
  <Accordion title="VaultGate peut-il remplacer mon pare-feu existant?">
    Oui, VaultGate est une solution de pare-feu de nouvelle génération complète avec IDS/IPS, VPN, filtrage web et plus encore.
  </Accordion>

  <Accordion title="Combien d'utilisateurs VPN peuvent se connecter simultanément?">
    - VaultGate 1000: 50 utilisateurs VPN simultanés
    - VaultGate 5000: 250 utilisateurs VPN simultanés
    - VaultGate 10000: 1 000 utilisateurs VPN simultanés
    - VaultGate Enterprise: Illimité
  </Accordion>

  <Accordion title="Supportez-vous le SD-WAN?">
    Oui, les modèles VaultGate 5000 et supérieurs incluent des capacités SD-WAN avec sélection de chemin intelligente et routage sensible aux applications.
  </Accordion>
</AccordionGroup>

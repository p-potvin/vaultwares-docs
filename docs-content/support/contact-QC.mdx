## Avant de contacter le support

Pour nous aider à vous assister rapidement, veuillez avoir les informations suivantes prêtes :

<Steps>
  <Step title="Informations sur le produit">
    - Nom et modèle du produit
    - Version du logiciel ou du micrologiciel
    - Clé de licence ou numéro de série
  </Step>
  <Step title="Informations sur le système">
    - Système d'exploitation et version
    - Spécifications matérielles
    - Configuration réseau (le cas échéant)
  </Step>
  <Step title="Détails du problème">
    - Description détaillée du problème
    - Étapes pour reproduire le problème
    - Messages ou codes d'erreur
    - Captures d'écran ou journaux (si disponibles)
  </Step>
  <Step title="Évaluation de l'impact">
    - Combien d'utilisateurs sont affectés ?
    - Est-ce que cela bloque des opérations critiques ?
    - Quand le problème a-t-il commencé ?
  </Step>
</Steps>

## Collecter les informations de diagnostic

### Windows

```powershell
# Generate diagnostic report
vw-diagnostic --full-report --output C:\vw-diagnostic.zip

# Collect system logs
Get-EventLog -LogName Application -Source VaultWares* |
  Export-Csv C:\vw-logs.csv

# Check service status
Get-Service | Where-Object {$_.Name -like "VaultWares*"}
```

### macOS

```bash
# Generate diagnostic report
vw-diagnostic --full-report --output ~/vw-diagnostic.zip

# Collect system logs
log show --predicate 'subsystem == "com.vaultwares"' --last 24h > ~/vw-logs.txt

# Check service status
launchctl list | grep com.vaultwares
```

### Linux

```bash
# Generate diagnostic report
sudo vw-diagnostic --full-report --output /tmp/vw-diagnostic.zip

# Collect system logs
sudo journalctl -u vaultwares* --since "24 hours ago" > /tmp/vw-logs.txt

# Check service status
sudo systemctl status vaultwares*
```

## Soumettre un ticket de support

<Steps>
  <Step title="Se connecter au portail de support">
    Visitez [portal.vaultwares.com/support](https://portal.vaultwares.com/support) et connectez-vous avec vos identifiants de compte.
  </Step>
  <Step title="Créer un nouveau ticket">
    Cliquez sur « Nouveau ticket » et sélectionnez la catégorie appropriée :
    - Problème technique
    - Licence ou facturation
    - Demande de fonctionnalité
    - Demande générale
  </Step>
  <Step title="Fournir des détails">
    Remplissez tous les champs obligatoires et joignez des fichiers de diagnostic si disponibles.
  </Step>
  <Step title="Définir la priorité">
    - **Critique** : Arrêt du système, perte de données, violation de sécurité
    - **Élevée** : Fonctionnalité majeure altérée
    - **Moyenne** : Problème de fonctionnalité mineur
    - **Faible** : Question générale ou demande d'amélioration
  </Step>
  <Step title="Soumettre et suivre">
    Soumettez votre ticket et suivez les progrès via les notifications par courriel ou le portail de support.
  </Step>
</Steps>

## Processus d'escalade

Si vous n'êtes pas satisfait du support que vous recevez :

<Steps>
  <Step title="Contacter votre ingénieur de support">
    Discutez de vos préoccupations avec l'ingénieur de support assigné.
  </Step>
  <Step title="Demander une escalade">
    Demandez que votre ticket soit escaladé à un ingénieur senior ou un chef d'équipe.
  </Step>
  <Step title="Contacter le gestionnaire de support">
    Envoyez un courriel à support-manager@vaultwares.com avec votre numéro de ticket.
  </Step>
  <Step title="Escalade exécutive">
    Pour les problèmes critiques non résolus, contactez : escalations@vaultwares.com
  </Step>
</Steps>

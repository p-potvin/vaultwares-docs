## Vue d'ensemble

VaultWares publie régulièrement des mises à jour logicielles et de micrologiciel. Ces mises à jour peuvent inclure des correctifs de sécurité, des corrections de bogues, de nouvelles fonctionnalités et des améliorations de performance. Rester à jour avec les mises à jour est une partie importante de votre posture de sécurité.

## Canaux de mise à jour

| Channel | Description | Recommandé pour |
|---------|-------------|-----------------|
| **Stable** | Versions rigoureusement testées, publiées environ trimestriellement | Tous les environnements de production |
| **LTS** | Soutien à long terme avec engagement de correctifs de sécurité de 5 ans | Environnements réglementés avec des fenêtres de changement peu fréquentes |
| **Security** | Correctifs hors bande pour les vulnérabilités de sécurité critiques | Tous les environnements — à prioriser immédiatement |
| **Beta** | Tests de pré-publication | Laboratoires et environnements non de production seulement |

## Comment vérifier les mises à jour

```bash
# Check all managed products for available updates
vw-admin updates check --all --format table

# Output:
# Product          Current     Available   Channel  Severity   Release date
# VaultCrypt       3.1.2       3.2.0       Stable   Security   2024-11-01
# VaultAccess      3.1.5       3.2.0       Stable   Feature    2024-11-01
# VaultBackup      3.0.8       3.2.0       Stable   Feature    2024-11-01
# VaultMonitor     3.1.1       3.2.0       Stable   Feature    2024-11-01
# VaultDrive FW    3.1.2       3.2.0       Stable   Security   2024-10-15
# VaultHSM FW      5.4.1       5.5.0       Stable   Feature    2024-10-20
# VaultGate FW     4.2.3       4.3.0       Stable   Security   2024-11-05

# Subscribe to security advisories via email
vw-admin updates subscribe \
  --email security@example.com \
  --channels security,stable \
  --products all
```

## Procédures de mise à jour logicielle

### Notifications de mise à jour automatiques

```bash
# Configure automatic update checks and email notifications
vw-admin updates configure \
  --check-frequency daily \
  --notify-email security@example.com \
  --auto-download true \
  --auto-install false \  # Review before installing in production
  --channel stable
```

### Mise à jour manuelle (Linux)

```bash
# Update VaultCrypt
sudo apt-get update && sudo apt-get install --only-upgrade vaultcrypt-enterprise

# Update VaultAccess
sudo apt-get update && sudo apt-get install --only-upgrade vaultaccess

# Verify installed versions
vaultcrypt --version
vw-admin --version
vw-monitor --version
vw-backup --version
```

### Mise à jour progressive Kubernetes

```bash
# Update all VaultWares deployments to latest stable
for deployment in vaultaccess vaultbackup vaultmonitor; do
  kubectl set image deployment/$deployment \
    $deployment=registry.vaultwares.com/$deployment:3.2.0 \
    -n vaultwares
  kubectl rollout status deployment/$deployment -n vaultwares
done
```

## Avis de sécurité

Abonnez-vous pour recevoir des avis de sécurité par courriel :
[portal.vaultwares.com/security-advisories](https://portal.vaultwares.com/security-advisories)

Les mises à jour de sécurité critiques (CVSS 7.0+) sont annoncées via :
- Courriel à tous les propriétaires de produits enregistrés
- Notification sur la bannière du portail
- Flux RSS : `https://security.vaultwares.com/advisories.rss`

VaultWares suit la [divulgation responsable](https://security.vaultwares.com/disclosure) et se coordonne avec les chercheurs par [security@vaultwares.com](mailto:security@vaultwares.com).

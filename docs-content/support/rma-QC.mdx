## Vue d'ensemble

Le processus d'Autorisation de Retour de Marchandise (RMA) est utilisé lorsque vous devez retourner un produit matériel VaultWares défectueux ou endommagé pour le remplacement ou la réparation sous garantie, ou pour une élimination en fin de vie. Ce guide vous explique le processus complet de RMA.

## Quand faire une demande de RMA

Faites une demande de RMA si :

- Un produit matériel présente un défaut confirmé couvert par la garantie (voir [Informations sur la garantie](/support/warranty))
- Un appareil est arrivé endommagé (DOA — Dead on Arrival)
- Vous devez retourner un appareil pour une élimination/destruction sécurisée et recevoir un certificat de destruction
- Vous échangez un appareil contre un modèle ou une capacité différent (contactez les ventes pour les échanges)

<Note>
  Avant de faire une demande de RMA, ouvrez un ticket de support afin que notre équipe puisse confirmer le défaut et émettre un numéro de RMA. Les colis sans numéro de RMA valide peuvent être refusés ou retournés à vos frais.
</Note>

## Processus de RMA étape par étape

<Steps>
  <Step title="Ouvrir un ticket de support">
    Contactez le support VaultWares pour diagnostiquer le problème et confirmer la couverture de la garantie :

    - **Portail** : [portal.vaultwares.com/support](https://portal.vaultwares.com/support)
    - **Courriel** : [support@vaultwares.com](mailto:support@vaultwares.com)
    - **Téléphone** : Disponible pour les clients des plans de support Gold et Platinum

    Incluez dans votre ticket :
    - Numéro de série de l'appareil (imprimé sur l'étiquette de l'appareil)
    - Date d'achat et preuve d'achat (numéro de commande ou facture)
    - Description détaillée du défaut
    - Sortie de diagnostic : `vw-admin diag collect --output diag.tar.gz`
  </Step>

  <Step title="Recevoir l'autorisation de RMA">
    Une fois que le défaut est confirmé et que la couverture de la garantie est vérifiée, vous recevrez :

    - **Numéro de RMA** (format : `RMA-YYYYMMDD-NNNN`)
    - **Instructions de RMA** incluant l'adresse d'expédition de retour
    - **Étiquette d'expédition de retour prépayée** (pour les réparations sous garantie — service de garantie standard)
    - **Délai de résolution prévu**

    Les numéros de RMA sont valides pendant 30 jours à compter de leur émission. Contactez le support pour prolonger si nécessaire.
  </Step>

  <Step title="Effectuer une effacement cryptographique avant de retourner (OBLIGATOIRE)">
    Avant de retourner tout matériel VaultWares, vous **devez** effectuer un effacement cryptographique pour détruire tout le matériel de clé et les données de l'appareil. Cela protège votre organisation et est requis pour maintenir votre posture de sécurité.

    <Warning>
      Retourner un appareil sans effectuer d'effacement cryptographique peut exposer des données sensibles. VaultWares ne sera pas responsable des données qui resteraient sur les appareils retournés.
    </Warning>

    ```bash
    # Crypto-erase a VaultDrive
    vw-storage crypto-erase \
      --device /dev/sdb \
      --confirm \
      --audit-log "RMA return - RMA-20241115-0042"

    # Remote crypto-erase a managed VaultDrive
    vw-storage remote-crypto-erase \
      --serial VD-ENT-00247 \
      --confirm \
      --reason "RMA-20241115-0042"

    # Erase a VaultHSM (requires SO credentials)
    vw-hsm factory-reset \
      --so-pin <so-pin> \
      --confirm \
      --audit-log "RMA return - RMA-20241115-0043"
    ```

    Si l'appareil ne peut pas être mis sous tension ou est physiquement endommagé :
    - VaultWares peut effectuer une destruction physique et fournir un **Certificat de destruction**
    - Mentionnez dans votre demande de RMA que l'appareil ne peut pas être effacé avant le retour
  </Step>

  <Step title="Emballer l'appareil">
    Un emballage adéquat prévient les dommages pendant le transport et protège les fonctionnalités anti-manipulation du matériel VaultWares.

    **Exigences d'emballage** :
    - Utilisez l'**emballage d'origine VaultWares** si disponible. Sinon, utilisez une boîte solide de taille similaire avec au moins 5 cm de mousse ou de papier bulle sur tous les côtés.
    - **Ne pas** emballer les appareils directement dans des copeaux de rembourrage — ils peuvent endommager l'appareil et interférer avec les fonctionnalités de détection de manipulation.
    - Pour les appareils VaultHSM : les rails de rack et les accessoires n'ont pas besoin d'être retournés sauf si spécifiquement demandé.
    - Incluez une **copie imprimée du numéro de RMA** à l'intérieur de la boîte.
    - Écrivez le **numéro de RMA** à l'extérieur de la boîte.

    <Warning>
      Collez l'étiquette d'expédition prépayée à l'extérieur de la boîte, et non à la boîte originale du fabricant. Ne masquez aucune étiquette sur la boîte intérieure.
    </Warning>
  </Step>

  <Step title="Expédier le colis">
    Utilisez l'**étiquette d'expédition prépayée** fournie avec votre autorisation de RMA. Déposez le colis au point de transport désigné.

    Pour les colis qui n'utilisent pas l'étiquette prépayée (par exemple, retours non garantis), expédiez à l'adresse fournie dans votre autorisation de RMA. VaultWares recommande d'utiliser un service d'expédition traçable avec assurance pour les articles de grande valeur.

    Après l'expédition, mettez à jour votre ticket de support avec le **numéro de suivi**.
  </Step>

  <Step title="Suivre votre RMA">
    Surveillez le statut de votre RMA à [portal.vaultwares.com/support/rma](https://portal.vaultwares.com/support/rma) ou via le CLI :

    ```bash
    # Check RMA status
    vw-admin rma status --rma-number RMA-20241115-0042

    # Output:
    # RMA number:   RMA-20241115-0042
    # Status:       In transit to VaultWares
    # Tracking:     1Z999AA10123456784
    # Received:     Pending
    # Resolution:   Replacement shipment
    # ETA:          2024-11-22 (estimated)
    ```
  </Step>
</Steps>

## Types de résolution de RMA

| Résolution | Description | Délai |
|---|---|---|
| **Remplacement avancé** | L'appareil de remplacement est expédié avant la réception du retour (nécessite une retenue de carte de crédit) | Jour ouvrable suivant |
| **Remplacement standard** | L'appareil de remplacement est expédié après réception et inspection de l'unité défectueuse | 5 à 10 jours ouvrables après réception |
| **Réparation** | L'appareil est réparé et retourné | 10 à 15 jours ouvrables après réception |
| **Remboursement** | Disponible dans les 30 jours suivant l'achat pour les unités DOA | 5 à 10 jours ouvrables après réception |
| **Certificat de destruction** | Destruction physique avec certification écrite | 5 jours ouvrables après réception |

## Remplacement avancé

Le remplacement avancé expédie un nouvel appareil immédiatement pendant que vous retournez l'unité défectueuse.

- Nécessite une **carte de crédit valide** enregistrée (facturée uniquement si l'unité défectueuse n'est pas retournée dans les 30 jours)
- Disponible pour VaultDrive Pro/Enterprise, VaultHSM PCIe/Network et VaultScan
- Demandez un remplacement avancé lors de l'ouverture de votre ticket de support

## Expédition et logistique

| Situation | Coût d'expédition |
|---|---|
| Défaut sous garantie (dans la période de garantie) | VaultWares paie l'expédition de retour |
| DOA dans les 30 jours suivant l'achat | VaultWares paie l'expédition de retour |
| Retour ou échange non garanti | Le client paie l'expédition de retour |
| Expédition de remplacement avancé | VaultWares paie l'expédition de sortie |

VaultWares expédie les remplacements via FedEx Priority, UPS Next Day Air ou transporteur équivalent selon la région. Une signature est requise à la livraison pour les produits HSM.

## RMA internationaux

Pour les clients en dehors des États-Unis continentaux :

- Des étiquettes d'expédition de retour internationales sont fournies pour les réclamations sous garantie
- La documentation douanière est gérée par VaultWares — ne déclarez pas de valeur commerciale sur le formulaire de douane au-delà de ce que VaultWares fournit
- Prévoyez 5 à 10 jours ouvrables supplémentaires pour les RMA internationaux
- Contactez [support@vaultwares.com](mailto:support@vaultwares.com) pour confirmer la disponibilité régionale du remplacement avancé

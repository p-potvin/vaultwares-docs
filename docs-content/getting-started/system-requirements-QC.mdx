## Exigences matérielles

### Spécifications minimales

<AccordionGroup>
  <Accordion title="Périphériques de stockage chiffré">
    - **Interface**: USB 3.0 ou supérieur
    - **Alimentation**: Alimenté par USB (aucune alimentation externe requise)
    - **Température de fonctionnement**: 0°C à 45°C
    - **Température de stockage**: -20°C à 85°C
  </Accordion>

  <Accordion title="Modules de sécurité matériels (HSM)">
    - **Format**: PCIe x4 ou connecté au réseau
    - **Alimentation**: 25W maximum
    - **Refroidissement**: Refroidissement actif requis pour les unités montées en rack
    - **Réseau**: Gigabit Ethernet (pour les HSM réseau)
  </Accordion>

  <Accordion title="Périphériques biométriques">
    - **Interface**: USB 2.0 ou supérieur
    - **Alimentation**: Alimenté par USB
    - **Résolution du capteur**: 500 DPI minimum
    - **Température de fonctionnement**: 10°C à 35°C
  </Accordion>

  <Accordion title="Appareils de sécurité réseau">
    - **Espace en rack**: 1U ou 2U montable en rack
    - **Alimentation**: Bloc d'alimentation redondant double (100-240V CA)
    - **Réseau**: Double Gigabit Ethernet minimum
    - **Stockage**: 256GB SSD minimum
  </Accordion>
</AccordionGroup>

## Exigences logicielles

### Compatibilité du système d'exploitation

<Tabs>
  <Tab title="Windows">
    - Windows 10 (64 bits) ou ultérieur
    - Windows Server 2016 ou ultérieur
    - .NET Framework 4.8 ou ultérieur
    - 4 Go de RAM minimum (8 Go recommandé)
    - 500 Mo d'espace disque disponible
  </Tab>

  <Tab title="macOS">
    - macOS 11 (Big Sur) ou ultérieur
    - 4 Go de RAM minimum (8 Go recommandé)
    - 500 Mo d'espace disque disponible
    - Privilèges d'administrateur pour l'installation
  </Tab>

  <Tab title="Linux">
    - Ubuntu 20.04 LTS ou ultérieur
    - Red Hat Enterprise Linux 8 ou ultérieur
    - CentOS 8 ou ultérieur
    - Debian 11 ou ultérieur
    - 4 Go de RAM minimum (8 Go recommandé)
    - 500 Mo d'espace disque disponible
    - Accès root pour l'installation
  </Tab>
</Tabs>

### Exigences réseau

- **Bande passante**: 10 Mbps minimum (100 Mbps recommandé)
- **Ports**: HTTPS (443), SSH (22), ports personnalisés pour l'accès API
- **Pare-feu**: Accès HTTPS sortant pour la validation des licences et les mises à jour
- **DNS**: Résolution DNS fonctionnelle requise

### Compatibilité du navigateur

Pour les interfaces de gestion basées sur le web :

- Google Chrome 90 ou ultérieur
- Mozilla Firefox 88 ou ultérieur
- Microsoft Edge 90 ou ultérieur
- Safari 14 ou ultérieur

## Exigences de base de données

Pour les déploiements d'entreprise avec gestion centralisée :

- **PostgreSQL**: 12 ou ultérieur
- **MySQL**: 8.0 ou ultérieur
- **Microsoft SQL Server**: 2017 ou ultérieur
- **Oracle Database**: 19c ou ultérieur

## Prise en charge de la virtualisation

Le logiciel VaultWares est compatible avec :

- VMware vSphere 7.0 ou ultérieur
- Microsoft Hyper-V (Windows Server 2016 ou ultérieur)
- KVM (noyau 4.15 ou ultérieur)
- Conteneurs Docker (pour le déploiement de microservices)

<Warning>
  Les modules de sécurité matériels (HSM) nécessitent un accès matériel direct et ne peuvent pas être virtualisés. Utilisez le passage PCIe ou des HSM connectés au réseau pour les environnements virtualisés.
</Warning>

## Prise en charge des plateformes cloud

- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)
- Environnements cloud privés

## Exigences supplémentaires

### Pour l'intégration API

- TLS 1.2 ou ultérieur
- Certificats SSL/TLS valides
- Système de gestion des clés API
- Capacités d'analyse JSON

### Pour les fonctionnalités de conformité

- Protocole de temps réseau (NTP) pour l'horodatage précis
- Serveur Syslog pour la journalisation centralisée
- Capacités d'intégration SIEM

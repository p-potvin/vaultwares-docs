## Avant de commencer

Assurez-vous d'avoir lu les [exigences système](/getting-started/system-requirements) avant de procéder à l'installation.

<Check>
  Vous aurez besoin de privilèges administrateur ou root pour installer les produits VaultWares.
</Check>

## Installation matérielle

### Périphériques de stockage chiffrés

<Steps>
  <Step title="Connecter l'appareil">
    Branchez le périphérique de stockage chiffré sur un port USB 3.0 disponible de votre ordinateur.
  </Step>
  <Step title="Installer les pilotes">
    Les pilotes de l'appareil s'installeront automatiquement sur Windows et macOS. Pour Linux, exécutez :
    ```bash
    sudo apt-get install vaultwares-storage-drivers
    ```
  </Step>
  <Step title="Initialiser l'appareil">
    Lancez le Gestionnaire de stockage VaultWares et suivez l'assistant d'initialisation pour définir votre mot de passe maître.
  </Step>
  <Step title="Vérifier le chiffrement">
    Vérifiez l'état de l'appareil dans le Gestionnaire de stockage pour confirmer que le chiffrement est actif.
  </Step>
</Steps>

### Modules de sécurité matériels (HSM)

<Tabs>
  <Tab title="HSM PCIe">
    <Steps>
      <Step title="Éteindre le serveur">
        Éteignez complètement votre serveur et débranchez-le de l'alimentation.
      </Step>
      <Step title="Installer la carte HSM">
        Insérez la carte HSM dans un emplacement PCIe x4 ou supérieur disponible.
      </Step>
      <Step title="Mettre sous tension et installer le logiciel">
        Démarrez le serveur et installez le logiciel de gestion de l'HSM :
        ```bash
        sudo ./vaultwares-hsm-installer.sh
        ```
      </Step>
      <Step title="Initialiser l'HSM">
        Exécutez la commande d'initialisation :
        ```bash
        vw-hsm init --admin-pin <your-pin>
        ```
      </Step>
    </Steps>
  </Tab>

  <Tab title="HSM réseau">
    <Steps>
      <Step title="Monter l'appareil dans le rack">
        Installez l'HSM réseau dans votre baie de serveurs et connectez les câbles d'alimentation.
      </Step>
      <Step title="Connecter les câbles réseau">
        Connectez les deux ports Ethernet pour la redondance.
      </Step>
      <Step title="Accéder à l'interface web">
        Naviguez vers l'adresse IP par défaut : `https://192.168.1.100`
      </Step>
      <Step title="Terminer l'assistant de configuration">
        Suivez l'assistant de configuration basé sur le web pour configurer les paramètres réseau et initialiser l'HSM.
      </Step>
    </Steps>
  </Tab>
</Tabs>

### Périphériques biométriques

<Steps>
  <Step title="Connecter l'appareil">
    Branchez le lecteur biométrique sur un port USB.
  </Step>
  <Step title="Installer le logiciel">
    Téléchargez et exécutez l'installeur de la suite biométrique VaultWares depuis le portail client.
  </Step>
  <Step title="Enrôler les utilisateurs">
    Lancez le Gestionnaire biométrique et enrôlez les empreintes digitales ou les profils faciaux des utilisateurs.
  </Step>
  <Step title="Tester l'authentification">
    Vérifiez que les utilisateurs enrôlés peuvent s'authentifier avec succès.
  </Step>
</Steps>

## Installation logicielle

### Installation Windows

<Steps>
  <Step title="Télécharger l'installeur">
    Téléchargez le dernier installeur Windows depuis le [portail client](https://portal.vaultwares.com).
  </Step>
  <Step title="Exécuter l'installeur">
    Double-cliquez sur `VaultWares-Setup.exe` et suivez l'assistant d'installation.
  </Step>
  <Step title="Entrer la clé de licence">
    Lorsque vous y êtes invité, entrez votre clé de licence reçue par courriel.
  </Step>
  <Step title="Terminer l'installation">
    Cliquez sur Terminer et redémarrez votre ordinateur si on vous le demande.
  </Step>
</Steps>

### Installation macOS

<Steps>
  <Step title="Télécharger l'installeur">
    Téléchargez `VaultWares.dmg` depuis le portail client.
  </Step>
  <Step title="Monter et installer">
    Ouvrez le fichier DMG et faites glisser VaultWares dans votre dossier Applications.
  </Step>
  <Step title="Accorder les permissions">
    Allez dans Préférences Système > Sécurité et confidentialité et autorisez VaultWares à fonctionner.
  </Step>
  <Step title="Activer la licence">
    Lancez VaultWares et entrez votre clé de licence.
  </Step>
</Steps>

### Installation Linux

<Tabs>
  <Tab title="Debian/Ubuntu">
    ```bash
    # Add VaultWares repository
    wget -qO - https://repo.vaultwares.com/gpg.key | sudo apt-key add -
    echo "deb https://repo.vaultwares.com/apt stable main" | sudo tee /etc/apt/sources.list.d/vaultwares.list

    # Update and install
    sudo apt-get update
    sudo apt-get install vaultwares

    # Start the service
    sudo systemctl start vaultwares
    sudo systemctl enable vaultwares
    ```
  </Tab>

  <Tab title="RHEL/CentOS">
    ```bash
    # Add VaultWares repository
    sudo wget https://repo.vaultwares.com/rpm/vaultwares.repo -O /etc/yum.repos.d/vaultwares.repo

    # Install
    sudo yum install vaultwares

    # Start the service
    sudo systemctl start vaultwares
    sudo systemctl enable vaultwares
    ```
  </Tab>
</Tabs>

### Déploiement Docker

```bash
# Pull the latest image
docker pull vaultwares/security-suite:latest

# Run the container
docker run -d \
  --name vaultwares \
  -p 8443:8443 \
  -v /var/vaultwares/data:/data \
  -v /var/vaultwares/config:/config \
  -e LICENSE_KEY=your-license-key \
  vaultwares/security-suite:latest
```

## Configuration initiale

### Configurer le compte administrateur

<Steps>
  <Step title="Lancer VaultWares">
    Ouvrez l'application VaultWares ou naviguez vers l'interface web.
  </Step>
  <Step title="Créer le compte admin">
    Entrez votre courriel d'administrateur et créez un mot de passe fort (minimum 12 caractères).
  </Step>
  <Step title="Activer l'authentification à deux facteurs">
    Scannez le code QR avec votre application d'authentification et entrez le code de vérification.
  </Step>
  <Step title="Configurer les politiques de sécurité">
    Définissez les politiques de mot de passe, les délais de session et les contrôles d'accès.
  </Step>
</Steps>

### Se connecter à votre infrastructure

<CodeGroup>
```bash Active Directory
vw-config ad-connect \
  --domain example.com \
  --server dc1.example.com \
  --admin-user administrator
```

```bash LDAP
vw-config ldap-connect \
  --host ldap.example.com \
  --port 636 \
  --base-dn "dc=example,dc=com" \
  --bind-dn "cn=admin,dc=example,dc=com"
```

```bash SAML
vw-config saml-setup \
  --idp-url https://idp.example.com/saml \
  --entity-id https://vaultwares.example.com \
  --cert /path/to/idp-cert.pem
```
</CodeGroup>

## Vérifier l'installation

Exécutez l'outil de diagnostic pour vérifier votre installation :

```bash
vw-diagnostic --full-check
```

Résultat attendu :
```
✓ License valid
✓ Database connection successful
✓ Encryption modules loaded
✓ Network connectivity verified
✓ All services running

Installation Status: READY
```

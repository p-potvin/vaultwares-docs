# Recettes du jour (exemples)

Cette page sert à montrer, **en images**, c’est quoi une *recette* (un workflow).

## Comment lire ça (sans être tech)

- Chaque **boîte** = une étape.
- Chaque **flèche** = “après ça, on fait ça”.
- Le but : tu vois que n’importe quel hobby/métier peut devenir une recette.

<Tip>
  Ces recettes viennent de `C:\Users\Administrator\Desktop\gemini-daily-flow.md`. On les utilise comme **exemples** pour inspirer la bibliothèque de workflows dans `vault-flows`.
</Tip>

---

## Jour 1 — Observation d’oiseaux + identification

```mermaid
flowchart LR
  A["Dossier: nouvelles photos"] --> B["Lire les infos (date/lieu)"]
  B --> C["IA (vision): propose l’espèce + confiance"]
  D["Notes de l’humain: où/quoi/combien"] --> E["Assembler l’info"]
  C --> E
  E --> F{"Tu confirmes?"}
  F -->|Oui| G["Renommer + classer les fichiers"]
  F -->|Non| H["Corriger l’espèce (manuel)"]
  H --> G
  G --> I["Ajouter au journal (JSON)"]
  G --> J["Journal lisible (Markdown)"]
```

## Jour 2 — Planificateur de levain (pain)

```mermaid
flowchart TB
  A["Entrées: température, force du levain, heure de cuisson"] --> B["Calcul: vitesse de fermentation"]
  B --> C["Timeline: nourrir → mélanger → repos → façonnage → pousse → cuisson"]
  C --> D["Si la temp change: recalculer les temps"]
  D --> E{"Décision requise?"}
  E -->|Oui| F["Humain: 'c’est prêt? (volume/texture)'"]
  E -->|Non| G["Continuer"]
  F --> G
  G --> H["Rappels/notifications"]
  G --> I["Feuille imprimable"]
```

## Jour 4 — “Cueillette sécuritaire” (plantes / champignons)

```mermaid
flowchart TB
  A["Photos: détails (dessous, tige, etc.)"] --> B["Checklist: habitat + saison + notes"]
  B --> C["IA (vision): candidats + confiance"]
  C --> D["Règles: drapeaux rouges / sosies toxiques"]
  D --> E{"Confiance assez haute?"}
  E -->|Non| F["Refuser: 'ne pas manger' + notes"]
  E -->|Oui| G{"Revue humaine"}
  G -->|OK| H["Cataloguer + rappel sécurité"]
  G -->|Pas sûr| F
  H --> I["Exporter: journal perso / format communauté"]
```

## Jour 6 — Stratégie: générateur de recettes (anti‑répétition)

```mermaid

## Jour 10 — Script‑to‑Lux (éclairage de théâtre)

  C["Notes du metteur en scène"] --> D["Analyser l’ambiance"]

  H --> J["Export: PDF + CSV"]
```

## Pourquoi c’est important pour Vault Flows

Ces exemples montrent que “workflow” ne veut pas dire “programmation”.
C’est juste une façon de rendre une tâche **claire**, **répétable**, et **facile à partager**.

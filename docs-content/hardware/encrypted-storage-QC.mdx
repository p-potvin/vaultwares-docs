## Aperçu

Les périphériques de stockage chiffrés VaultWares offrent un chiffrement AES-256 basé sur le matériel pour la sécurité des données portables. Toutes les opérations de chiffrement et de déchiffrement se déroulent à l'intérieur du processeur sécurisé de l'appareil, garantissant que vos données n'existent jamais dans un état non chiffré sur l'ordinateur hôte.

## Fonctionnalités clés

### Chiffrement matériel
- **Chiffrement AES-256-XTS**: Norme de chiffrement de qualité militaire
- **Processeur sécurisé**: Puce de chiffrement dédiée, isolée de l'hôte
- **Aucun logiciel requis**: Fonctionne sans installation de pilotes ni d'applications
- **Protection contre les attaques par force brute**: Verrouillage automatique après des tentatives infructueuses

### Fonctionnalités de sécurité
- **Protection par mot de passe**: Mots de passe alphanumériques jusqu'à 64 caractères
- **Authentification à deux facteurs**: Prise en charge TOTP optionnelle (modèles Enterprise)
- **Mode autodestruction**: Effacement de la clé cryptographique sur commande
- **Détection de manipulation**: Alertes d'intrusion physique et verrouillage automatique
- **Mode lecture seule**: Protège les données sensibles en écriture

### Capacités de gestion
- **Gestion à distance**: Contrôle centralisé via la console de gestion VaultWares
- **Application des politiques**: Complexité des mots de passe, paramètres de délai d'expiration, contrôles d'accès
- **Journalisation des audits**: Journaux complets des accès et des opérations
- **Géorepérage**: Restrictions d'accès basées sur la localisation (modèles Enterprise)

## Configuration et utilisation

### Configuration initiale

<Steps>
  <Step title="Connecter l'appareil">
    Branchez le VaultDrive sur un port USB disponible.
  </Step>
  <Step title="Définir le mot de passe maître">
    Lors de la première utilisation, on vous demandera de créer un mot de passe maître (8 à 64 caractères).
  </Step>
  <Step title="Enrôler les données biométriques (si applicable)">
    Pour les modèles biométriques, scannez votre empreinte digitale en suivant les instructions à l'écran.
  </Step>
  <Step title="Accéder à votre lecteur">
    Le volume chiffré sera monté automatiquement après une authentification réussie.
  </Step>
</Steps>

### Utilisation quotidienne

```bash
# The device appears as a standard removable drive
# Windows: Assigned drive letter (e.g., E:)
# macOS: Mounted in /Volumes/VaultDrive
# Linux: Mounted in /media/username/VaultDrive

# Copy files normally
cp sensitive-data.pdf /Volumes/VaultDrive/

# Safely eject when done
# macOS
diskutil eject /Volumes/VaultDrive

# Linux
sudo umount /media/username/VaultDrive
```

### Gestion des mots de passe

<CodeGroup>
```bash Change password
# Using VaultWares CLI
vw-storage change-password --device /dev/sdb

# Or use the web interface
# Navigate to: https://localhost:8443/storage/manage
```

```bash Reset device
# Factory reset (erases all data)
vw-storage factory-reset --device /dev/sdb --confirm

# This action is irreversible
```

```bash Enable read-only mode
# Protect against accidental modifications
vw-storage set-readonly --device /dev/sdb --enable
```
</CodeGroup>

## Gestion en entreprise

### Déploiement centralisé

```bash
# Deploy configuration to multiple devices
vw-admin deploy-config \
  --policy corporate-standard \
  --devices device-list.txt \
  --enforce-password-complexity \
  --require-2fa \
  --audit-logging enabled
```

## Conformité et certifications

<AccordionGroup>
  <Accordion title="FIPS 140-2/140-3 Niveau 3">
    Tous les modèles VaultDrive sont certifiés FIPS 140-2 ou 140-3 Niveau 3, répondant aux exigences suivantes :
    - Détection et réponse à la manipulation physique
    - Authentification basée sur l'identité
    - Gestion des clés cryptographiques
    - Entrée et sortie de clés sécurisées
  </Accordion>

  <Accordion title="Common Criteria EAL5+/EAL6+">
    Évalué et certifié de manière indépendante pour la fonctionnalité et l'assurance de sécurité.
  </Accordion>

  <Accordion title="Conformité RGPD">
    Le chiffrement matériel garantit que les exigences de protection des données sont respectées pour le stockage portable.
  </Accordion>

  <Accordion title="Conformité HIPAA">
    Respecte les mesures de protection techniques pour les informations de santé protégées électroniquement (ePHI).
  </Accordion>
</AccordionGroup>

## Dépannage

<AccordionGroup>
  <Accordion title="Appareil non reconnu">
    - Essayez un autre port USB
    - Assurez-vous d'utiliser un port USB 3.0 ou supérieur
    - Mettez à jour les pilotes USB de votre système
    - Testez sur un autre ordinateur pour isoler le problème
  </Accordion>

  <Accordion title="Mot de passe oublié">
    Contactez le support VaultWares avec une preuve de propriété. Note : Les données ne peuvent pas être récupérées sans le mot de passe en raison du chiffrement matériel.
  </Accordion>

  <Accordion title="Vitesses de transfert lentes">
    - Vérifiez que vous utilisez un port USB 3.0+
    - Vérifiez la qualité du câble (utilisez le câble inclus)
    - Désactivez temporairement l'analyse en temps réel de l'antivirus
    - Assurez-vous qu'il y a suffisamment d'espace libre sur l'appareil
  </Accordion>

  <Accordion title="Capteur biométrique ne fonctionne pas">
    - Nettoyez le capteur avec un chiffon doux et sec
    - Ré-enregistrez votre empreinte digitale
    - Assurez-vous que votre doigt est sec et propre
    - Mettez à jour le micrologiciel de l'appareil via VaultWares Manager
  </Accordion>
</AccordionGroup>

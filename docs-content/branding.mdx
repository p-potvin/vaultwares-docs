# VaultWares Brand Guidelines

**Last updated**: April 24, 2026  
**Version**: 1.0  
**Single source of truth**: This file. All agents and developers must reference it.

## Core Identity

- **Mission**: Cybersecurity and data privacy products built 100% from Free Open-Source Software & Hardware.
- **Tagline**: "Your data. Your rules. VaultWares."
- **Tone**: Professional yet approachable • Privacy-first • Technically precise • Quietly confident • No hype.
- **Forbidden language**: "just", "simply", "easy" (unless factually proven), "military-grade", competitor product names.

## Visual Standards

- **Primary color**: `#0A2540` (deep vault blue)
- **Accent color**: `#00D4FF` (cyber cyan)
- **Logo**: Use `/logo/vaultwares-logo.svg` (never stretch, recolor, or compress)
- **Typography**: Inter (headings & body), IBM Plex Mono (code blocks)

## Repository Header Template

Use in every `README.md`:

```markdown
<img src="https://raw.githubusercontent.com/p-potvin/vaultwares-docs/main/logo/vaultwares-logo.svg">

# Repository Name

**Part of the VaultWares Ecosystem** • <a href="https://docs.vaultwares.com">docs.vaultwares.com</a> • <a href="https://vaultwares.com">vaultwares.com</a>
```
